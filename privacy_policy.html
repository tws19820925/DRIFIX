DRIFIX プライバシーポリシー

DRIFIX は、船釣り向けの海況確認、釣り座提案、時合確認を支援するアプリです。 本ポリシーでは、アプリにおける情報の取得、利用、第三者サービスの利用、連絡先について説明します。

最終更新日: 2026年4月4日

1. 取得する情報

  • 位置情報: おおよその現在地、正確な現在地
  • アプリの操作情報: 画面遷移、魚種・釣法選択、保存操作など
  • 診断情報: クラッシュログ、診断情報、アプリ状態
  • 端末またはその他の ID: Firebase などの SDK が扱う識別子

2. 情報の利用目的

  • 海況、潮流、波、海水温、風向など本アプリの機能提供
  • 地点ごとの釣り座提案や表示内容の最適化
  • 障害解析、品質改善、将来の機能改善
  • 利用傾向の確認と操作導線の改善

3. 位置情報と外部送信

本アプリは、現在地表示や海況取得のために位置情報を利用します。位置情報の利用は、ユーザーの許可に基づいて行います。

また、取得した位置情報に応じて海況を取得するため、位置情報を海況 API や関連サービスへ送信する場合があります。 そのため、Google Play のデータ セーフティ申告上は、位置情報について「収集」および「共有」に該当する形で申告しています。

4. Firebase の利用

本アプリでは、主に以下の目的で Google Firebase を利用します。

  • Firebase Analytics: 操作イベントや利用傾向の把握
  • Firebase Crashlytics: クラッシュや不具合の解析

Analytics では、個人を特定しない形で操作イベントや利用傾向を記録する場合があります。 たとえば、地図タップ、魚種選択、釣法選択、保存操作などのイベントを確認します。

Crashlytics では、障害解析のため、現在のタブ、地点、魚種、釣法、日時、アプリ状態などの文脈情報を custom key として記録する場合があります。

5. 第三者サービス

本アプリは、次の第三者サービスまたは API を利用することがあります。

  • Google Maps Platform
  • Google Firebase
  • Open-Meteo
  • 海況関連 API
  • JAXA G-Portal / GCOM-C SST などの公開データ

各サービスの仕様変更に伴っては、それぞれの公開ポリシーもあわせてご確認ください。

6. データの削除について

現時点では、ユーザーごとの個別データ削除をアプリから直接リクエストする専用機能は提供していません。 Google Play のデータ セーフティ申告でも、その前提で回答しています。

7. 安全管理

本アプリでは、通信中のデータ保護に暗号化を利用します。これにより、Google Play のデータ セーフティ申告において 「データは安全に転送されます」と申告しています。
