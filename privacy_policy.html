DRIFIX プライバシーポリシー

DRIFIX（以下「本アプリ」）は、海況確認、釣り座提案、釣行計画支援を目的としたアプリです。 本ポリシーでは、本アプリにおける情報の取得、利用、第三者サービスの利用、保護方針について説明します。

最終更新日: 2026年4月4日

1. 取得する情報

  • 位置情報: おおよその現在地、正確な現在地
  • アプリの操作情報: 地点選択、魚種・釣法選択、お気に入り登録、釣り座確認操作など
  • 診断情報: クラッシュログ、診断情報、アプリの状態に関する技術情報
  • 端末またはその他の ID: Firebase などの SDK が取り扱う識別子

2. 情報の利用目的

  • 海況、潮流、海水温、地図表示など本アプリの機能提供
  • 地点ごとの釣り座提案や表示内容の最適化
  • 障害解析、安定性向上、品質改善
  • 利用傾向の把握による機能改善

3. 位置情報と外部送信

本アプリは、現在地表示や海況取得のために位置情報を利用します。位置情報の利用は、ユーザーの許可に基づいて行います。

また、選択地点や現在地に対応した海況を取得するため、位置情報を含む必要な情報を外部の海況 API や関連サービスへ送信する場合があります。 そのため、Google Play のデータ セーフティ申告上は、位置情報について「収集」と「共有」の両方を申告しています。

4. Firebase の利用

本アプリでは、Google Firebase を利用しています。主な用途は以下のとおりです。

  • Firebase Analytics: 操作イベントや利用傾向の把握
  • Firebase Crashlytics: クラッシュや不具合の解析

Analytics では、地図タップ、魚種選択、釣法選択、お気に入り登録、釣り座確認などの操作イベントを計測する場合があります。 ただし、個人を直接特定することを目的とした利用は行いません。

Crashlytics では、不具合解析のために、現在のタブ、選択中の地点、魚種、釣法、日時、アプリ版などの文脈情報を custom key として記録する場合があります。これらは障害調査や品質改善を目的として利用します。

5. 第三者サービス

本アプリは、以下の第三者サービスまたは API を利用することがあります。

  • Google Maps Platform
  • Google Firebase
  • Open-Meteo
  • 海況関連 API
  • JAXA G-Portal / GCOM-C SST などの公開データ

各サービスの取り扱いについては、それぞれの提供元ポリシーもあわせてご確認ください。

6. データの削除について

現時点では、ユーザーが自分のデータ削除をアプリ内から直接リクエストできる専用機能は提供していません。 Google Play のデータ セーフティ申告でも、その旨を申告しています。

7. 安全管理

本アプリでは、取得したデータの送信時に暗号化通信を利用します。これにより、Google Play のデータ セーフティ申告において 「データは送信中に暗号化される」と申告しています。
