プライバシーポリシー

アプリ名: FishingSeatAdvisor / DRIFIX
最終更新日: 2026年4月2日

FishingSeatAdvisor（以下、「本アプリ」）は、釣り座検討や海況確認を支援するため、 位置情報、端末内保存データ、外部 API 通信を利用します。本ポリシーは、本アプリにおける 情報の取扱いを説明するものです。

1. 取得または利用する情報

  • 位置情報: 現在地表示、地図表示、地点選択補助のために利用します。
  • ユーザー入力または選択情報: 代表海域、魚種、釣法、日時、地図上で選択した座標などを扱います。
  • 端末内保存情報: 最近使った条件、お気に入り条件、表示設定、直前のセッション状態を端末内に保存します。
  • 通信情報: 海況、天気、波、潮位、海水温などの取得のため、外部サービスに必要なリクエストを送信します。

2. 利用目的

  • 地図表示、地点選択、現在地周辺の確認
  • 釣り座候補、気象、潮汐、海況、海水温などの情報表示
  • お気に入り条件、最近使った条件、表示設定の保存と再利用
  • アプリの動作安定化、不具合調査、利用性改善

3. 外部サービスの利用

本アプリは以下の外部サービスまたはデータ提供元を利用する場合があります。

  • Google Maps SDK for Android
  • Google Play services
  • Cloud Run 経由の海水温取得 API
  • 気象、海況、潮汐、海水温などの各種データ提供元

これらのサービスでは、各提供元のプライバシーポリシーに基づいて情報が処理される場合があります。

4. 端末内保存について

本アプリは、セッション状態、表示設定、お気に入り条件、最近使った条件などを端末内の保存領域に保持します。 これらは主に利便性向上のために使用され、開発者が常時取得するものではありません。

5. 位置情報について

位置情報の利用は、端末上でユーザーが権限を許可した場合に限られます。位置情報を許可しない場合でも、 一部機能を除き、本アプリは代表海域選択や地図タップによって利用できます。

6. 第三者提供

法令に基づく場合を除き、開発者がユーザーの個人情報を第三者へ販売または提供することを目的として 収集することはありません。ただし、前記の外部サービス利用に伴い、通信に必要な範囲で情報が送信される場合があります。

7. 安全管理

本アプリは、秘密情報のリポジトリ管理除外、release 署名、難読化設定など、公開運用に必要な保護措置を整備しています。 ただし、通信環境や外部サービスの性質上、完全な安全性を保証するものではありません。

8. 改定

本ポリシーは、機能追加や法令改正に応じて更新されることがあります。重要な変更がある場合は、 アプリ内または公開ページ上で案内します。
